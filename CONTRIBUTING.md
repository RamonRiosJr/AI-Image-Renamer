# Contributing

We welcome contributions to the AI Image Renamer! Whether it's a bug fix, new feature, or documentation improvement, we appreciate your help.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Create a new branch** from `main` for your changes.
3. **Make your changes**. Ensure your code follows the existing style and is well-documented.
4. **Run locally**: Test your changes by running `npm run dev`.
5. **Submit a Pull Request**: Provide a clear description of your changes and why they are necessary.

## Quality Bar

- Ensure all PRs pass the Gitleaks CI scan.
- For UI changes, follow the existing Tailwind CSS and React architecture.
- Avoid committing secrets or private data.

Thank you for contributing!

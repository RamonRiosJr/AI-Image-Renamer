# Security Policy

## Supported Versions

We recommend staying up to date with the latest `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do NOT create a public issue. Instead, report it privately via GitHub's **Vulnerability Reporting** feature on the repository, or email the maintainer directly.

We will acknowledge your report within 48 hours and work to resolve the issue as quickly as possible.
